# ``ShellKit``

The <code>ShellKit</code> is the framework to support the development shell program.

## Overview

The <code>ShellKit</code> is the framework to support the development shell program.
The framework supports macOS and iOS. The software is written by Swift programming language with Xcode developer tool.
